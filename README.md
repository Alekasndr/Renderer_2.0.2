# Renderer_2.0.2

How to build this project:

$mkdir build<br />
$cd build<br />
$cmake .. -D sdl2_DIR:FILEPATH="Here should be path to SDL2"<br />
$cmake --build . --config Release<br />

This rendering engine is possible to render different models with different shader.
